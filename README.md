## 🌐 Live Preview

You can host this for free with **GitHub Pages**:
1. Go to your repo on GitHub
2. Settings → Pages → Source: `main` branch → `/` (root)
3. Your site will be live at `https://<your-username>.github.io/<repo-name>/`
